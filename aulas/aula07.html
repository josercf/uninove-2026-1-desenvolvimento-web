O que é .NET?

.NET é uma plataforma de desenvolvimento criada pela Microsoft. É open-source, multiplataforma (Windows, Linux, macOS) e suporta diversas linguagens, sendo C# a principal.

Componentes

  • .NET SDK — Ferramentas de desenvolvimento
  • CLR — Runtime que executa o código
  • BCL — Biblioteca de classes base
  • ASP.NET Core — Framework web

Por que .NET?

  • Alta performance
  • Ecossistema robusto
  • Grande mercado de trabalho
  • Suporte da Microsoft

C# é uma linguagem fortemente tipada. Diferente do JavaScript, você precisa declarar o tipo da variável.

Variáveis e Constantes

// Variáveis
string nome = "João";
int idade = 20;
double nota = 8.5;
bool aprovado = true;

// Constantes (não podem ser alteradas)
const double PI = 3.14159;
const string UNIVERSIDADE = "UNINOVE";

// Interpolação de strings
Console.WriteLine($"Aluno: {nome}, Idade: {idade}");
Console.WriteLine($"Nota: {nota:F1} - Aprovado: {aprovado}");

// Conversão de tipos
string textoNumero = "42";
int numero = int.Parse(textoNumero);
double valor = Convert.ToDouble("3.14");

Entrada de Dados pelo Console

// Lendo dados do usuário
Console.Write("Digite seu nome: ");
string nome = Console.ReadLine();

Console.Write("Digite sua idade: ");
int idade = int.Parse(Console.ReadLine());

Console.Write("Digite sua nota: ");
double nota = double.Parse(Console.ReadLine());

// Exibindo resultado
Console.WriteLine($"Olá, {nome}! Você tem {idade} anos.");
Console.WriteLine($"Sua nota é: {nota:F1}");

Console.ReadLine() sempre retorna uma string. Use int.Parse() ou Convert.ToInt32() para converter.

Exercício Prático 2

Calculadora em C#

  4. Calcule e exiba o resultado
  5. Trate a divisão por zero
  6. Desafio: Faça o programa repetir até o usuário digitar 0 para sair
