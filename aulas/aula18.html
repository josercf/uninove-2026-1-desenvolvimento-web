_Layout.cshtml — Template Mestre

O _Layout.cshtml é o template mestre que define a estrutura comum de todas as páginas (menu, rodapé, CSS, JS).

<!-- Views/Shared/_Layout.cshtml -->
<!DOCTYPE html>
<html>
<head>
    <title>@ViewData["Title"] - Minha App</title>
    <link rel="stylesheet" href="~/lib/bootstrap/css/bootstrap.min.css" />
    <link rel="stylesheet" href="~/css/site.css" />
</head>
<body>
    <nav>...</nav>
    <div class="container">
        @RenderBody()  <!-- Conteúdo da página -->
    </div>
    <footer>...</footer>
    @RenderSection("Scripts", required: false)
</body>
</html>

Usando o Layout nas Views

<!-- Views/Aluno/Index.cshtml -->
@{
    ViewData["Title"] = "Lista de Alunos";
}

<h2>Lista de Alunos</h2>
<!-- Este conteúdo será inserido no @RenderBody() -->
<table class="table table-striped">
    ...
</table>

@section Scripts {
    <script>
        // Scripts específicos desta página
        console.log("Página de alunos carregada");
    </script>
}

O _ViewStart.cshtml define o layout padrão: @{ Layout = "_Layout"; }

Partial Views

Partial Views são fragmentos reutilizáveis de HTML/Razor. Úteis para menus, cards, formulários repetidos.

Criar a Partial

<!-- Views/Shared/_CardAluno.cshtml -->
@model Aluno

<div class="card mb-3">
  <div class="card-body">
    <h5>@Model.Nome</h5>
    <p>@Model.Email</p>
    <p>@Model.Curso.Nome</p>
  </div>
</div>

Usar a Partial

<!-- Na View principal -->
@model List<Aluno>

<div class="row">
@foreach (var aluno in Model)
{
  <div class="col-md-4">
    @await Html
      .PartialAsync(
        "_CardAluno", aluno)
  </div>
}
</div>

Bootstrap — Navbar

<!-- No _Layout.cshtml -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <div class="container">
    <a class="navbar-brand" asp-controller="Home" asp-action="Index">
      Minha App
    </a>
    <div class="navbar-nav">
      <a class="nav-link" asp-controller="Aluno" asp-action="Index">
        Alunos
      </a>
      <a class="nav-link" asp-controller="Curso" asp-action="Index">
        Cursos
      </a>
      @if (User.Identity.IsAuthenticated)
      {
        <a class="nav-link" asp-controller="Auth" asp-action="Logout">
          Sair (@User.Identity.Name)
        </a>
      }
    </div>
  </div>
</nav>

Exercício Prático 1

Refatorar com Layout e Bootstrap

  1. Configure o _Layout.cshtml com Bootstrap
  2. Adicione uma Navbar com links para Alunos, Cursos e Login/Logout
  3. Adicione um Footer com informações do projeto
  4. Refatore todas as Views existentes para usar classes Bootstrap
  5. Use table-striped nas tabelas e btn nos botões

Exercício Prático 2

Dashboard com Partial Views

  1. Crie uma Partial View _CardAluno para exibir dados de um aluno em um card
  2. Crie uma Partial View _CardEstatistica para exibir um número e um título
  3. Crie uma página Dashboard que exiba: total de alunos, total de cursos, últimos 5 alunos cadastrados
  4. Use o grid do Bootstrap para organizar os cards
  5. Desafio: Adicione um alert de boas-vindas com o nome do usuário logado
