Flexbox — Layout em 1 Dimensão

Flexbox organiza elementos em uma única direção (linha ou coluna). Ideal para alinhar e distribuir espaço.

.container {
    display: flex;
    justify-content: space-between;  /* eixo principal */
    align-items: center;           /* eixo cruzado */
    gap: 20px;                       /* espaço entre itens */
}

CSS Grid — Layout em 2 Dimensões

Grid organiza elementos em linhas e colunas simultaneamente. Ideal para layouts complexos.

.grid {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;  /* 3 colunas */
    gap: 15px;
}

Formulários HTML

Formulários são o principal ponto de interação entre o usuário e a aplicação. Permitem enviar dados ao servidor.

<form action="/processar" method="POST">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>

    <button type="submit">Enviar</button>
</form>

action

URL para onde os dados serão enviados

method

GET (dados na URL) ou POST (dados no corpo)

Tipos de Input

TipoCódigoUso

E-mail<input type="email">Validação automática de e-mail
Senha<input type="password">Oculta caracteres digitados
Número<input type="number">Valores numéricos com setas

Outros Elementos de Formulário

<!-- Área de texto (múltiplas linhas) -->
<textarea name="mensagem" rows="4" cols="50"></textarea>

<!-- Lista suspensa -->
<select name="estado">
    <option value="">Selecione...</option>
    <option value="SP">São Paulo</option>
    <option value="RJ">Rio de Janeiro</option>
</select>

<!-- Atributos de validação -->
<input type="text" required minlength="3" maxlength="50">
<input type="number" min="1" max="100">

Exercício Guiado

Criando um Formulário de Contato

30 minutos
  1. Crie contato.html e contato.css
  2. Crie um <form> com method="POST"
  3. Adicione campos: Nome (text), Email (email), Assunto (select) e Mensagem (textarea)
  4. Use <label> para cada campo com o atributo for
  5. Adicione required nos campos obrigatórios
  6. No CSS, use Flexbox (flex-direction: column) para organizar os campos
  7. Estilize os inputs com padding, border-radius e uma borda suave
  8. Crie um botão de submit estilizado com cor de fundo e hover

Exercício Guiado (continuação)

Layout com Flexbox

20 minutos
  1. Crie layout.html com um <header>, <main> e <footer>
  2. No header, crie uma <nav> com 4 links usando Flexbox
  3. Use justify-content: space-between para distribuir os links
  4. No main, crie 3 cards lado a lado usando display: flex
  5. Cada card deve ter flex: 1 para ocupar espaço igual
  6. Adicione gap: 20px entre os cards
