Unidades de Medida

Absolutas vs. Relativas

Medidas Absolutas

Medidas absolutas são fixas e não mudam de acordo com o dispositivo. A mais comum é o pixel (px).

UnidadeNomeEquivalência
pxPixel1px = 1 ponto na tela
cmCentímetro1cm = 37.8px
mmMilímetro1mm = 3.78px
ptPonto1pt = 1.33px

Atenção: O pixel do CSS não é exatamente o pixel físico da tela. É um "pixel de referência" que pode variar conforme a densidade da tela.

Medidas Relativas

Medidas relativas são calculadas com base em outra unidade de referência, permitindo layouts mais flexíveis.

UnidadeReferênciaExemplo
emFont-size do elemento paiSe pai = 16px, 2em = 32px
remFont-size do <html> (raiz)Se raiz = 16px, 1.5rem = 24px
%Tamanho do elemento pai50% da largura do pai
vwLargura da viewport100vw = largura total da tela
vhAltura da viewport100vh = altura total da tela

Exemplo: em vs. rem

Usando em

.pai {
    font-size: 16px;
}
.filho {
    font-size: 2em;
    /* 2 × 16px = 32px */
    padding: 1em;
    /* 1 × 32px = 32px */
}

O em pode gerar efeito cascata!

Usando rem

html {
    font-size: 16px;
}
.elemento {
    font-size: 2rem;
    /* 2 × 16px = 32px */
    padding: 1rem;
    /* 1 × 16px = 16px */
}

O rem é sempre relativo à raiz!

Formulários HTML

Interação entre usuário e aplicação

Formulários HTML

Formulários são o principal ponto de interação entre o usuário e a aplicação web. Permitem que os usuários enviem dados para o servidor.

<form action="/processar" method="POST">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    <button type="submit">Enviar</button>
</form>

action define para onde os dados serão enviados. method define o método HTTP (GET ou POST).

Tipos de Input

TipoCódigoUso
Texto<input type="text">Nome, endereço
E-mail<input type="email">Validação automática
Senha<input type="password">Oculta caracteres
Número<input type="number">Valores numéricos
Data<input type="date">Seletor de data
Checkbox<input type="checkbox">Múltipla escolha
Radio<input type="radio">Escolha única

Outros Elementos de Formulário

<!-- Área de texto (múltiplas linhas) -->
<textarea name="mensagem" rows="4" cols="50"></textarea>

<!-- Lista suspensa -->
<select name="estado">
    <option value="SP">São Paulo</option>
    <option value="RJ">Rio de Janeiro</option>
    <option value="MG">Minas Gerais</option>
</select>

<!-- Botão de envio -->
<button type="submit">Enviar</button>

Exercício Prático 1

Página de Apresentação com Unidades de Medida

  1. Crie uma página HTML com um cabeçalho "Bem-vindo à Minha Página"
  2. Defina a largura da página como 800px e centralize
  3. Configure o cabeçalho com altura de 100px e texto centralizado
  4. Adicione um parágrafo com largura de 600px e margem superior de 20px
  5. Agora refaça o mesmo exercício substituindo px por em
  6. Compare os resultados ao redimensionar a janela do navegador

Exercício Prático 2

Formulário de Cadastro Completo

Crie um formulário de cadastro com os seguintes campos:

  1. Nome completo (text, obrigatório)
  2. E-mail (email, obrigatório)
  3. Senha (password, mínimo 6 caracteres)
  4. Data de nascimento (date)
  5. Gênero (radio buttons: Masculino/Feminino/Outro)
  6. Estado (select com pelo menos 5 opções)
  7. Interesses (checkboxes: Tecnologia, Esportes, Música, Leitura)
  8. Observações (textarea)
  9. Botões de Enviar e Limpar

Estilize o formulário com CSS externo usando unidades rem.
