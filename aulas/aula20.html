Projeto Final

Especificação e Critérios

Especificação do Projeto Final

Desenvolva uma aplicação web completa usando ASP.NET Core MVC com os seguintes requisitos obrigatórios:

Requisitos Obrigatórios

  1. Padrão MVC — Controllers, Models e Views organizados
  2. Banco de Dados MySQL com Entity Framework Core
  3. CRUD completo de pelo menos 2 entidades
  4. Relacionamento 1:N entre as entidades
  5. Autenticação — Registro, login e logout
  6. Validação com Data Annotations
  7. Layout com _Layout.cshtml e Bootstrap
  8. Pelo menos 1 funcionalidade com AJAX
