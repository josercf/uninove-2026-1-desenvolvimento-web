O que é JavaScript?

JavaScript é a linguagem de programação da Web. Enquanto HTML define a estrutura e CSS define a aparência, o JavaScript define o comportamento e a interatividade das páginas.

Características

  • Linguagem interpretada
  • Executada no navegador
  • Tipagem dinâmica
  • Baseada em eventos

O que pode fazer?

  • Validar formulários
  • Alterar conteúdo da página
  • Reagir a cliques e eventos
  • Fazer requisições ao servidor

Variáveis e Tipos de Dados

// Declaração de variáveis
let nome = "Maria";        // String
let idade = 25;              // Number
let ativo = true;            // Boolean
let notas = [8, 9, 7];       // Array
let endereco = null;         // Null

// Constante (não pode ser reatribuída)
const PI = 3.14159;

// Exibir no console
console.log(nome);           // "Maria"
console.log(typeof idade);   // "number"

Use let para variáveis que mudam e const para valores fixos. Evite var.

Operadores

Aritméticos

let a = 10, b = 3;
a + b   // 13 (soma)
a - b   // 7  (subtração)
a * b   // 30 (multiplicação)
a / b   // 3.33 (divisão)
a % b   // 1  (resto)

Comparação

a == b    // igual (valor)
a === b   // idêntico (valor+tipo)
a != b    // diferente
a > b     // maior que
a < b     // menor que
a >= b    // maior ou igual

Sempre use === (igualdade estrita) em vez de == para evitar conversões inesperadas.

Estruturas de Controle

Condicional (if/else)

let nota = 7;

if (nota >= 7) {
    console.log("Aprovado!");
} else if (nota >= 5) {
    console.log("Recuperação");
} else {
    console.log("Reprovado");
}

Repetição (for)

let frutas = ["Maçã", 
  "Banana", "Laranja"];

for (let i = 0; 
     i < frutas.length; 
     i++) {
    console.log(frutas[i]);
}

// Ou com for...of
for (let fruta of frutas) {
    console.log(fruta);
}

Manipulação do DOM

Alterando a página com JavaScript

O que é o DOM?

O DOM (Document Object Model) é a representação em árvore de todos os elementos HTML da página. JavaScript pode acessar e modificar qualquer elemento do DOM.

// Selecionar elementos
let titulo = document.getElementById("meuTitulo");
let paragrafo = document.querySelector(".destaque");
let itens = document.querySelectorAll("li");

// Modificar conteúdo
titulo.textContent = "Novo Título";
titulo.style.color = "blue";

// Reagir a eventos
titulo.addEventListener("click", function() {
    alert("Você clicou no título!");
});

Validação de Formulários com JS

<form id="meuForm">
    <input type="text" id="nome" placeholder="Seu nome">
    <input type="email" id="email" placeholder="Seu e-mail">
    <button type="submit">Enviar</button>
</form>

<script>
document.getElementById("meuForm")
  .addEventListener("submit", function(e) {
    let nome = document.getElementById("nome").value;
    if (nome.trim() === "") {
        e.preventDefault();
        alert("O nome é obrigatório!");
    }
});
</script>

Exercício Prático 1

Calculadora Simples

Exercício Prático 2

Validação do Formulário de Cadastro

Retome o formulário de cadastro da Aula 05 e adicione validações com JavaScript:

  1. Nome deve ter no mínimo 3 caracteres
