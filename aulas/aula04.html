Atributos HTML

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento.

<p class="destaque" id="intro">Texto com atributos</p>
<img src="foto.jpg" alt="Descrição" width="300">
<a href="https://uninove.br" target="_blank">Link</a>
TipoExemplosUso
Globaisclass, id, style, langAceitos por todas as tags
Específicossrc, href, alt, disabledApenas em determinadas tags

CSS

Cascading Style Sheets

O que é CSS?

CSS (Cascading Style Sheets) é a tecnologia usada para estilizar e desenhar o conteúdo HTML. Com CSS você pode alterar fontes, cores, tamanhos, espaçamentos, criar layouts e adicionar animações.

CSS é uma linguagem baseada em regras — você define grupos de estilo que devem ser aplicados a elementos específicos.

h1 {
    color: red;
    font-size: 32px;
}

/* seletor { propriedade: valor; } */

Sintaxe CSS

seletor {
    propriedade: valor;
    propriedade: valor;
}

Componentes

  • Seletor: aponta para o elemento HTML
  • Propriedade: o que será estilizado
  • Valor: como será estilizado
  • Chaves { }: delimitam as declarações

Exemplo

p {
    color: #333333;
    font-size: 18px;
    line-height: 1.6;
    margin-bottom: 10px;
}

Seletores CSS

SeletorSintaxeSeleciona
Tipo (tag)h1 { }Todos os <h1>
Classe.destaque { }Elementos com class="destaque"
ID#unico { }Elemento com id="unico"
Atributoa[href] { }Links que possuem href
Múltiplosh1, h2, .box { }Todos os listados

Regra: Classes podem ser reutilizadas em vários elementos. IDs devem ser únicos na página.

3 Formas de Adicionar CSS

1. Inline (no elemento)

<p style="color: red;">
  Texto vermelho
</p>

Evite! Mistura estrutura com estilo.

2. Interno (no <head>)

<style>
  p { color: red; }
</style>

3. Externo (arquivo .css)

<link rel="stylesheet"
  href="estilo.css">

Recomendado! Separação de responsabilidades.

Box Model

Todo elemento HTML é uma "caixa" composta por 4 camadas:

margin
border
padding
conteúdo

Largura total = content + padding + border + margin

Propriedades Comuns do CSS

Texto e Fonte

color: #333;
font-size: 16px;
font-family: Arial, sans-serif;
font-weight: bold;
text-align: center;
text-decoration: none;

Box Model

width: 300px;
height: 200px;
padding: 20px;
margin: 10px auto;
border: 1px solid #ccc;
background-color: #f0f0f0;

Exercício Prático 1

Estilizando a Página da Aula Anterior

  1. Abra o arquivo index.html da aula 03
  2. Crie um arquivo estilo.css na mesma pasta
  3. Vincule o CSS ao HTML usando <link>
  4. Defina a fonte do body como Arial
  5. Altere a cor do <h1> para azul escuro (#00274D)
  6. Adicione uma borda inferior ao <h1>
  7. Defina o background-color do body como #f5f5f5
  8. Centralize o conteúdo com max-width: 800px e margin: auto

Exercício Prático 2

Cartão de Visita Digital

Crie um cartão de visita digital usando HTML e CSS:

  1. Crie um <div> com classe cartao
  2. Largura de 400px, centralizado na página
  3. Borda arredondada (border-radius: 10px)
  4. Sombra (box-shadow: 0 4px 8px rgba(0,0,0,0.2))
  5. Padding de 30px
  6. Dentro do cartão: nome (h2), cargo (h3) e contato (p)
  7. Use classes para estilizar cada elemento
