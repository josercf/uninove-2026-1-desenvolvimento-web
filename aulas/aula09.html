A resposta correta é (a). O @ é o caractere especial do Razor. Exemplo: @ViewBag.Nome ou @DateTime.Now.

Condicionais no Razor

@{
    var nota = 8.5;
}

@if (nota >= 7) {
    <div class="alert alert-success">
        Aprovado com nota @nota
    </div>
} else if (nota >= 5) {
    <div class="alert alert-warning">
        Recuperação
    </div>
} else {
    <div class="alert alert-danger">
        Reprovado
    </div>
}

No Razor, o @if alterna entre C# e HTML automaticamente. As chaves { } delimitam o bloco.

Switch no Razor

@{
    var diaSemana = DateTime.Now.DayOfWeek;
}

@switch (diaSemana) {
    case DayOfWeek.Monday:
        <p>Segunda-feira — Início da semana!</p>
        break;
    case DayOfWeek.Friday:
        <p>Sexta-feira — Quase fim de semana!</p>
        break;
    default:
        <p>Hoje é @diaSemana</p>
        break;
}

Use switch quando tiver muitas condições sobre o mesmo valor. Para poucas condições, if/else é mais simples.

Arrays e List<T> em C#

Array (tamanho fixo)

string[] frutas = { 
    "Maçã", 
    "Banana", 
    "Laranja" 
};

// Acessar por índice
frutas[0]; // "Maçã"

List<T> (dinâmica)

var frutas = new List<string>
{
    "Maçã", 
    "Banana"
};

// Adicionar item
frutas.Add("Laranja");
frutas.Count; // 3

Prefira List<T> na maioria dos casos. Possui métodos úteis: Add(), Remove(), Find().

Laços de Repetição no Razor

// Controller: passa uma lista via ViewBag
public IActionResult Index()
{
    ViewBag.Produtos = new List<string>
    {
        "Notebook", "Mouse", "Teclado", "Monitor"
    };
    return View();
}

@* View: exibe a lista com foreach *@
<ul>
    @foreach (var produto in ViewBag.Produtos)
    {
        <li>@produto</li>
    }
</ul>

Exercício Guiado

Catálogo de Produtos com Condicionais

35 minutos
  1. Crie Controllers/ProdutoController.cs
  2. No método Index(), crie uma List<string> com 5 nomes de produtos
  3. Crie uma List<double> com os preços correspondentes
  4. Passe ambas as listas via ViewBag
  5. Na View, use @for para criar uma tabela com #, Nome e Preço
  6. Use @if para colorir de verde os produtos com preço < R$ 100
  7. Adicione um contador no final: "Total: X produtos"
