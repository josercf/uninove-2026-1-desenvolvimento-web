# Desenvolvimento Web — UNINOVE 2026.1

Material didático da disciplina **Desenvolvimento Web** do curso de graduação da UNINOVE, semestre 2026.1.

## Sobre a Disciplina

A disciplina aborda os fundamentos do desenvolvimento web, desde a estrutura da internet e tecnologias front-end (HTML, CSS, JavaScript) até o desenvolvimento back-end com **ASP.NET Core MVC**, **C#**, **Entity Framework Core** e **MySQL**.

## Metodologia

As aulas seguem a metodologia de **sala de aula invertida**:

- **Antes da aula:** O aluno estuda o material indicado (leitura, vídeo ou pesquisa)
- **Durante a aula:** Discussão, esclarecimento de dúvidas e exercícios práticos
- **Após a aula:** Consolidação com exercícios complementares

## Como Usar os Slides

Os slides estão em formato HTML e podem ser abertos diretamente no navegador. Navegação:

- **Seta direita** ou **clique** → próximo slide
- **Seta esquerda** → slide anterior
- **Home** → primeiro slide
- **End** → último slide
